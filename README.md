
# [Celeste Wiki](https://felipebstefens.github.io/Celeste-Wiki/index.html)

This is a school project that we need to use HTML and CSS to create a simple website in one week. 

We created a Celeste Wiki, that shows information of the game for each phase.

# How we make it

As a Website, we used HTML and CSS to make the design. 

We got a 3d Model of the Celeste Mountain so the user can interact and select the phase. So we use Blender to get and organize the model.

To apply, we used JS with ThreeJS. The model doesn't have color, so we use a background video, lights (point, directional and ambient light) and Celeste Soundtrack to make it immersive.
